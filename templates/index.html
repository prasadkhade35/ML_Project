ℹ️ Model Information

  • Champion: Random Forest Regressor — scikit-learn ensemble of 200 trees
  • Validated with Group K-Fold (k=3) — whole experiments held out per fold
  • Target generalisation R² 0.90–0.97 — balanced bias-variance trade-off
  • CO₂ is predicted first; CO and C₂H₄ use CO₂ as an auxiliary feature
  • Dataset: Berkowicz & Żukowski (2020), 1 266 FTIR observations, 90 experiments

🔌 REST API — Programmatic Access

Send a POST request to /api/predict with a JSON body: 

curl -X POST http://127.0.0.1:5000/api/predict \
     -H 'Content-Type: application/json' \
     -d '{
           "temperature":  700,
           "bed_type":     "Inert",
           "polymer_type": "PE",
           "mass_mg":      150,
           "time_s":       20
         }'

  "status": "ok",
